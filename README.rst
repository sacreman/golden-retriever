`Quick Example`_
----------------------------

install golden-retriever using pip

.. code-block:: none

    pip install golden-retriever

set the url in  ~/.golden-retriever.yaml to match the DalmatinerDB front end address and port

.. code-block:: none

    ---
    url: http://dfe.dataloop.io:8080
